# Рублёвые цены в каталоге

## Цель

Показывать клиенту ориентир стоимости до открытия формы заявки. На карточках и строках каталога цена размещается справа в формате `от 2 030 ₽`.

## Расчёт

- Сервер получает текущий USDT/RUB тем же механизмом, который используется для котировок.
- Для каждого сервиса выбирается минимальный активный тариф.
- Ориентир считается существующей формулой: `USD × (USDT/RUB × 1,05) × 1,30`, с округлением вверх до 10 ₽.
- Каталог возвращает рассчитанный ориентир вместе с тарифами. Браузер не рассчитывает деньги самостоятельно.
- Ориентир не является котировкой. Точная цена фиксируется только после выбора тарифа и создания 30-минутной котировки.

## Интерфейс

- В популярных карточках и полном списке справа показывается `от N ₽` лаймовым цветом.
- Для сервиса без активных автоматических тарифов показывается `Расчёт менеджером`.
- Цена не перекрывает название или описание: текстовая область может сжиматься, цена остаётся отдельным элементом без переноса цифр.
- На узких экранах цена переносится ниже названия и выравнивается по левому краю.

## Ошибки

- Если курс временно недоступен, каталог продолжает работать и показывает `Расчёт менеджером` вместо ошибки страницы.
- Истёкший старше 24 часов курс не используется.

## Проверка

- Один runnable-check проверяет выбор минимального тарифа и рублёвый ориентир.
- Браузерная проверка охватывает популярные карточки, полный список и мобильную ширину.
